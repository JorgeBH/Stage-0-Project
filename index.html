How the web is composed and how it works:



- The world wide web is composed by a huge number of documents and to be specific: HTML Documents

- The components to make the WWW work are: HTML Docs, Protocol (HTTP), Servers (in which are hosted this Docs), Web Browser, and the Internet itself.

What is the Hypertext Markup Language and how it works:


To be specific the HTML is constituted by a structure made of TAGS and those may have or may have not whats called attributes.

This tags allow to give format and more readable structure to documents. This is what makes possible to give an HTML document dynamism.
